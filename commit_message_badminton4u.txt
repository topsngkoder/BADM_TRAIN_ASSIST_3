Добавлено поле для ссылки на профиль Badminton4u

- Добавлено поле badminton4u_url в форму добавления игрока
- Добавлено поле badminton4u_url в форму редактирования игрока
- Обновлены функции handleAddPlayer и handleEditPlayer для обработки нового поля
- Обновлена функция openEditModal для заполнения поля при редактировании
- Добавлено отображение ссылки на профиль Badminton4u в карточке игрока
- Добавлены стили для ссылки на профиль Badminton4u

Это изменение позволяет пользователям добавлять ссылки на профили игроков в Badminton4u.ru, 
что может быть полезно для быстрого доступа к официальным рейтингам и статистике игроков.
